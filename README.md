# جزارة صاحب السعادة 🥩 (Saheb El-Saada Butchery)

تطبيق ويب متكامل وحديث لمحل وجزارة "صاحب السعادة" بعرض المنتجات الكندوز والجاموسي، مقاطع الفيديو الحية، عروض الباكجات، وإدارة الطلبات المباشرة عبر الواتساب ولوحة تحكم شاملة للآدمن.

## 🚀 المميزات الأساسية
- **عرض المنتجات والأقسام**: أقسام متنوعة (كندوز، جاموسي، مفروم، بلدي، عروض خاصة) مع أسعار ديناميكية وحساب الأوزان.
- **عروض وصور حية**: دعم رفع واستعراض الصور والفيديوهات ببروتوكولات المعاينة المباشرة (MediaUploader مع دعم السحب والإسقاط، النسخ واللصق، والرفع المحلي).
- **طلب مباشر عبر WhatsApp**: تحويل السلة وتفاصيل الطلب والعنوان تلقائياً برابط واتساب مجهز.
- **لوحة تحكم إدارية (Admin Dashboard)**:
  - حماية بالرمز السري (PIN) مع إمكانية تعديله وحفظه في أي وقت.
  - إحصائيات سريعة للطلبات والإيرادات.
  - إدارة كاملة للمنتجات، العروض، وفيديوهات الواجهة والمعرض.
  - إدارة الطلبات وإمكانية حذف طلب معين أو مسح الطلبات الملغية أو كافة الطلبات دفعة واحدة.
- **دعم الشاشات والهواتف (RTL & Responsive)**: تصميم داكن فخم مخصص للأجهزة الذكية والشاشات الكبيرة.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **React 19** + **TypeScript**
- **Vite** (أداة التطوير والتجميع السريعة)
- **Tailwind CSS v4** (للتنسيقات)
- **Lucide React** (للايقونات العصرية)
- **Motion / Framer Motion** (للمؤثرات والحركات)
- **Canvas Confetti** (للاحتفالات والتأثيرات)

## 📦 التشغيل المحلي (Local Development)

```bash
# 1. تثبيت الحزم والمكتبات
npm install

# 2. تشغيل سيرفر التطوير المحلي
npm run dev
```

التطبيق سيعمل على: `http://localhost:3000`

## 🚀 النشر على Vercel (Vercel Deployment)

المشروع مجهز بالكامل بملف `vercel.json` للرفع على **Vercel** بضغط زر واحدة:

1. ارفع الكود على حسابك في GitHub.
2. ادخل على [Vercel](https://vercel.com) واضغط **New Project**.
3. اختر الـ Repository الخاص بك.
4. اتترك الإعدادات الافتراضية (Framework Preset: **Vite**).
5. اضغط **Deploy**.
